Pocket Fridge Web APP

I worked with a team of 7 to create a web APP using agile practices in order to reduce the amount of food waste. Some features of the Pocket Fridge is that it can track what users have in their fridge, remind users when an item is about to expire, find recipes and add recipes, and statistics about their food usage and waste. We used Google Firebase to store our data, which allows users to add household members to the same account. We used HTML, CSS and Javascript to create a functional web APP. We used Figma and Miro to brainstorm and design our APP. We also used React and XCode. I was the main designer and worked on full stack tasks.
